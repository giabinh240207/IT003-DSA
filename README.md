## 🌌 About This Repository

This repository serves as an extensive archive and training ground for **IT003 (Data Structures & Algorithms)**. It goes beyond merely passing assignments; the core philosophy here is to construct **fast, optimal, and strictly clean code** that mirrors competitive programming standards. 

Every algorithm is treated as an optimization problem where theoretical mathematical bounds meet practical implementation.

## ⚡ Key Highlights
- **Algorithmic Rigor:** Solutions heavily focus on reducing time and space complexity, striving for strict $\mathcal{O}(N \log N)$ or $\mathcal{O}(1)$ performance where mathematically viable.
- **Deep Architecture:** Native implementations of core structures (Advanced Trees, Graphs, Hash Maps) without over-reliance on standard libraries to build a stronger foundational understanding.
- **Memory Management:** Careful attention to pointers, references, and avoiding memory leaks in C++.

## 🛠️ Development Environment
- **Language:** Standard C++ (`C++17` / `C++20`)
- **Compilation Setup:** Designed for seamless execution in Unix-based terminal environments (like Zsh on macOS).
